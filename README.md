# Team Task Manager

A full-stack team collaboration and task management platform built using React, Node.js, Express, PostgreSQL, and Prisma.

---

# Live Demo

Frontend:
https://ethra-ai-assessment.vercel.app/

Backend:
https://ethra-ai-assessment.onrender.com/

---

# Features

## Authentication
- User Signup
- User Login
- JWT Authentication
- Role-based Access Control

## Project Management
- Create Projects
- View Projects
- Add Team Members

## Task Management
- Create Tasks
- Edit Tasks
- Delete Tasks
- Update Task Status
- Task Priorities

## Dashboard
- Total Projects
- Total Tasks
- Completed Tasks
- Overdue Tasks
- Analytics Dashboard

---

# Tech Stack

## Frontend
- React.js
- Vite
- Tailwind CSS

## Backend
- Node.js
- Express.js
- Prisma ORM
- JWT Authentication

## Database
- PostgreSQL

# Installation

## Clone Repository

```bash
https://github.com/SourabhSaharan/ethra.ai_assessment
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
